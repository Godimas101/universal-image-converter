# 🤝 Contributing to SE Image Converter

Thanks for helping improve `universal-image-converter`.

---

## 🐞 Bugs and feature requests

Please open a GitHub issue for:

- conversion bugs or crashes
- DDS output problems
- LCD text conversion issues
- UI or packaging bugs
- documentation fixes or feature ideas

Screenshots and sample input files are helpful when they can be shared safely.

---

## 🔀 Pull requests

PRs are welcome.

Please:

1. keep changes focused and reviewable
2. update docs when user-visible behavior changes
3. avoid committing local-only files, build outputs, or scratch assets
4. explain any conversion logic changes clearly in the PR description

---

## 👀 Review flow

This repo uses code ownership and auto-assignment so **`@Godimas101`** is requested on pull requests and tagged as the default maintainer contact.
